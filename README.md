# planning-poker-nodejs
Planning poker - one of three app for scrum poker during PGS Workshops.

# Starting

- To copy config template use:
    ```bash
    npm run install
    ```

- To start app

    ```bash
    npm start
    ```
- if you want to run with DEBUG mode 

    ```bash
    npm run debug
    ```
